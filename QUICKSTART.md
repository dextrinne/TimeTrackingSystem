# БЫСТРЫЙ СТАРТ - Quick Start Guide

## 🚀 Запуск приложения / Launching the Application

### Способ 1: Двойной клик (Double Click) ✨
Просто дважды кликните на `run.bat` и приложение запустится!

**Windows Users:**
- Simply double-click `run.bat` file
- The application will start automatically!

### Способ 2: Командная строка (Command Line)
```bash
python app.py
```

### Способ 3: PowerShell
```powershell
python .\app.py
```

## 📚 Основные операции / Basic Operations

### 1️⃣ Добавление сотрудника / Add Employee
1. Нажмите кнопку **"➕ Добавить сотрудника"** / Click **"➕ Add Employee"**
2. Заполните форму:
   - **ФИО** / Full Name
   - **Должность** / Position
   - Количество дней / Number of days for each category
3. Нажмите **"Сохранить"** / Click **"Save"**

### 2️⃣ Редактирование / Edit Employee
1. **Дважды кликните** на нужного сотрудника / Double-click the employee
2. Измените данные / Update the information
3. Нажмите **"Сохранить"** / Click **"Save"**

### 3️⃣ Удаление / Delete Employee
1. Выберите сотрудника / Select the employee
2. Нажмите **"🗑️ Удалить сотрудника"** / Click **"🗑️ Delete"**
3. Подтвердите удаление / Confirm deletion

### 4️⃣ Экспорт в Excel / Export to Excel
1. Выберите месяц и год / Select month and year
2. Нажмите **"📊 Экспортировать в Excel"** / Click **"📊 Export to Excel"**
3. Выберите папку для сохранения / Choose save location
4. ✓ Готово! / Done!

## 📂 Импорт существующих данных / Import Existing Data

### Если у вас уже есть файлы табелей / If you have existing timesheet files:

```bash
python import_excel.py "табель-январь2026.xls"
```

**Or use the batch file:**
```
import.bat "табель-январь2026.xls"
```

**Result:** Данные будут загружены в приложение / Data will be loaded into the application

## ❓ Часто задаваемые вопросы / FAQ

### Q: Где хранятся мои данные?
**A:** В файле `data/employees.json` в папке приложения

### Q: Как восстановить удаленные данные?
**A:** Найдите резервную копию файла `data/employees.json` или переимпортируйте из старого Excel файла

### Q: Можно ли экспортировать в другие форматы?
**A:** В текущей версии поддерживается только Excel (.xlsx)

### Q: Как изменить список сотрудников?
**A:** Используйте кнопки "Добавить" и "Удалить" в приложении

### Q: Почему приложение не запускается?
**A:** Убедитесь, что:
1. Python установлен
2. Библиотеки установлены: `pip install -r requirements.txt`
3. Вы находитесь в правильной директории

## 🔧 Решение проблем / Troubleshooting

### Ошибка: "ModuleNotFoundError"
```bash
# Установите зависимости
pip install -r requirements.txt
```

### Ошибка: "Permission denied"
- Убедитесь, что у вас есть права на запись в папку
- Запустите от администратора (Run as Administrator)

### Данные не сохраняются
- Проверьте наличие папки `data`
- Убедитесь, что у вас есть права на запись
- Нажмите "💾 Сохранить данные" вручную

### Excel файл не импортируется
- Проверьте формат файла (.xls или .xlsx)
- Убедитесь, что файл содержит нужные колонки
- Попробуйте пересохранить файл в формате .xlsx
