## Способ 2: PostgreSQL (для полного функционала)

### 1. Установите PostgreSQL 14+
Скачайте с https://www.postgresql.org/download/

### 2. Создайте базу данных
```bash
psql -U postgres -f database/init_db.sql
```

### 3. Установите зависимости
```bash
pip install psycopg2-binary openpyxl
```

### 4. Обновите config.py
```python
DB_CONFIG = {
    'host': 'localhost',
    'port': 5432,
    'database': 'time_tracking',
    'user': 'postgres',
    'password': 'ваш_пароль'
}
```

### 5. Запустите
```bash
python main.py
```

## Возможные проблемы

### "ModuleNotFoundError: No module named 'psycopg2'"
Это нормально — приложение автоматически переключается на SQLite.

### "База данных недоступна"
Запустите `python database/init_sqlite.py` для создания локальной БД.

### Ошибка при вводе даты
Формат даты: `ГГГГ-ММ-ДД` (например: `2026-03-15`)

## Инструкция пользователя

1. **Запустите приложение**: `python main.py`
2. **Войдите в систему** (логин/пароль из таблицы выше)
3. **Добавьте сотрудников**: вкладка «Сотрудники» → «Добавить»
4. **Создайте табель**: вкладка «Табели» → «Создать табель»
5. **Сформируйте дни**: кнопка «Сформировать дни»
6. **Добавьте документы**: вкладка «Документы» → «Добавить документ»
7. **Посмотрите отчёт**: вкладка «Отчёты» → «Сформировать отчёт»
