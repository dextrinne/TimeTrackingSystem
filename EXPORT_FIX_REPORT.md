# ИСПРАВЛЕНИЕ ЭКСПОРТА В EXCEL

## 🔴 ПРОБЛЕМА
Экспорт в Excel выглядел неправильно - данные не соответствовали структуре шаблона.

## ✅ РЕШЕНИЕ

### 1. **Исправлены коды дней в DAY_CODES**
```
было: 'Р': 'Р'         (неправильный код)
стало: 'Р': 'ДО'       (правильный код: День Обыкновенный)
```

### 2. **Структура экспорта в Excel приведена в соответствие с шаблоном**

Каждый сотрудник занимает 2 строки:

**Первая строка (часовые данные):**
- Колонка 0: № п/п
- Колонка 1: ФИО
- Колонка 13: Должность
- Колонки 22+: Часы для рабочих дней (4, 8 и т.д.) или коды выходных (В, О, Б, П)

**Вторая строка (коды для отчёта):**
- Колонка 13: Ставка
- Колонки 22+: Коды дней (ДО, В, ОТ, Б, ПР)

### 3. **Правильная трансформация кодов**
```
Внутренний код → Код Excel
'Р' (рабочий)    → 'ДО' (День Обыкновенный)
'В' (выходной)   → 'В'
'О' (отпуск)     → 'ОТ'
'Б' (больничный) → 'Б'
'П' (прочее)     → 'ПР'
```
