## Общая структура проекта

```diff
TimeTrackingSystem/
│
├── app.py                      # Точка входа в приложение
│
├── core/                       # Ядро приложения
│   ├── __init__.py
│   ├── time_table_app.py       # Главный контроллер
│   ├── data_manager.py         # Управление данными
│   └── template_exporter.py    # Экспорт в Excel
│
├── ui/                         # Пользовательский интерфейс
│   ├── __init__.py
│   ├── main_window.py          # Главное окно с вкладками
│   ├── dialogs.py              # Диалоговые окна
│   └── widgets.py              # Виджеты календаря
│
├── frontend/                   # Стили и конфигурация
│   ├── __init__.py
│   ├── styles.py               # Цвета, шрифты, типы дней
│   └── config.py               # Настройки приложения
│
├── data/
│   └── employees.json          # База сотрудников
│
├── help (no use)/              # Файлы для помощи в разработке (не участвуют в работе кода)
│
└── archive/                    # Архив табелей (генерируется автоматически при работе)
    └── *.json                  # Сохраненные табели
