# Система управления табелями учета рабочего времени

**Версия 2.0** - Теперь с интерактивным календарем! 📅

## Описание
Приложение для удобного заполнения табелей учета рабочего времени сотрудников с интерактивным календарем и экспортом в Excel. Позволяет быстро и наглядно выбирать тип дня для каждой даты месяца.

## Возможности
- ✅ **Интерактивный календарь** - кликайте на каждый день месяца
- ✅ **Цветовое кодирование** - рабочие дни, выходные, отпуск, больничные и т.д.
- ✅ **Автоматический подсчет** - количество дней считается в реальном времени
- ✅ **Навигация по месяцам** - переходите между месяцами с кнопками
- ✅ **Удобное управление** - добавление, редактирование, удаление сотрудников
- ✅ **Импорт данных** - загрузка существующих табелей из Excel
- ✅ **Экспорт в Excel** - сохранение в красиво оформленные файлы
- ✅ **Автосохранение** - данные сохраняются автоматически
- ✅ **Русский интерфейс** - полная поддержка русского языка

## Требования
- Python 3.7+
- Библиотеки: openpyxl, pandas, xlrd

## Установка

### 1. Установка Python
Убедитесь, что у вас установлен Python 3.7 или выше.

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Запуск приложения
```bash
python app.py
```

## Использование

### Основной интерфейс
1. **Добавить сотрудника** - Нажмите кнопку "➕ Добавить сотрудника" для добавления нового сотрудника
2. **Редактировать** - Дважды кликните на строку сотрудника для редактирования
3. **Удалить** - Выберите сотрудника и нажмите "🗑️  Удалить сотрудника"
4. **Сохранить** - Данные сохраняются автоматически, но можно нажать "💾 Сохранить данные"
5. **Экспортировать** - Нажмите "📊 Экспортировать в Excel" для сохранения в файл

#### Как использовать календарь:
1. **Выберите тип дня** - Нажмите кнопку (Рабочий день, Выходной и т.д.)
2. **Кликайте на дни** - Они изменят цвет согласно выбранному типу
3. **Переходите между месяцами** - Используйте кнопки ◀ и ▶
4. **Видите подсчет** - Внизу показано "Выбранные дни: ..."
5. **Сохраняйте** - Нажмите "Сохранить"

#### Цветовое кодирование:
- 🟢 **Зеленый** - Рабочий день
- 🔵 **Голубой** - Выходной
- 🟡 **Золотой** - Отпуск
- 🟣 **Розовый** - Больничный
- 🟣 **Сливовый** - Прочее
- ⚪ **Белый** - Не выбран

## Формат экспортируемого Excel файла

Экспортируемый файл содержит:
- Заголовок с названием табеля
- Период (месяц и год)
- Дату составления
- Таблицу с данными сотрудников
- Итоговую строку с суммами по каждой колонке

## Сохранение данных

Все данные о сотрудниках автоматически сохраняются в файл `data/employees.json`. Этот файл создается автоматически при первом запуске приложения.

## Решение проблем

### Приложение не запускается
1. Убедитесь, что Python установлен: `python --version`
2. Установите зависимости: `pip install -r requirements.txt`
3. Проверьте, что вы находитесь в директории проекта

### Ошибка при экспорте в Excel
1. Убедитесь, что Excel не открывает файл с тем же названием
2. Проверьте, что у вас есть прав на запись в указанную папку
3. Убедитесь, что openpyxl установлена: `pip install openpyxl`

### Данные не сохраняются
1. Проверьте, что папка `data` имеет права на запись
2. Нажмите "💾 Сохранить данные" вручную

## Примеры использования

### Экспорт табеля за январь 2026
1. Выберите "январь" в выпадающем списке месяцев
2. Убедитесь, что год установлен на 2026
3. Заполните данные всех сотрудников
4. Нажмите "📊 Экспортировать в Excel"
5. Выберите место для сохранения файла
